# 👩‍💼 HR Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **HR Analytics Dashboard** built using **Power BI**.
The dashboard focuses on employee attrition, workforce trends, and performance metrics to help organizations make data-driven HR decisions.

---

## 🎯 Objective

The main objective of this project is to:

* Analyze employee attrition patterns
* Understand factors affecting employee retention
* Evaluate department-wise performance
* Identify key areas for HR improvement

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* Microsoft Excel

---

## 📊 Key KPIs

* **Total Employees:** 50K
* **Overall Attrition Rate:** 50.21%
* **Average Hourly Rate:** 115.43
* **Average Work-Life Balance:** 2.50

---

## 📈 Dashboard Insights

### 🔹 Attrition Analysis

* Overall attrition rate is **around 50%**, indicating high employee turnover
* Attrition remains relatively consistent across income levels

### 🔹 Department-wise Attrition

* Departments like **Research & Development and Software** show slightly higher attrition rates
* Highlights need for better retention strategies in specific departments

### 🔹 Work-Life Balance

* Average work-life balance is **2.5**, suggesting moderate employee satisfaction
* Lower balance may contribute to higher attrition

### 🔹 Promotion Impact

* Attrition rate remains high even with increased years since last promotion
* Indicates promotion alone may not be sufficient for retention

### 🔹 Job Role Insights

* Roles like **Sales Executive and Laboratory Technician** show varied work-life balance
* Helps HR identify roles needing attention
